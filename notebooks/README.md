# Notebooks

This directory contains Jupyter notebooks for data exploration, processing, and evaluation of the Complaint Analysis for Financial Services project.

## Notebooks Overview

- `compliant-EDA.ipynb`: Exploratory Data Analysis (EDA) of customer complaints data, including visualizations and insights into complaint patterns.
- `chuncking-and-embedding.ipynb`: Notebook for text chunking and generating embeddings for the complaint narratives.
- `RAG-evaluation.ipynb`: Evaluation of the Retrieval-Augmented Generation (RAG) system performance on complaint queries.
